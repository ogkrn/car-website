# Mahindra Thar Images

Place your Mahindra Thar images in this folder with the following naming convention:

- `thar-1.jpg` - Front view
- `thar-2.jpg` - Side view
- `thar-3.jpg` - Interior view
- `thar-4.jpg` - Dashboard view
- `thar-5.jpg` - Rear view
- `thar-6.jpg` - Engine view or additional angle

## Recommended Image Specifications:
- Format: JPG or PNG
- Dimensions: 800x600 pixels (minimum)
- Aspect Ratio: 4:3
- File Size: Less than 500KB per image

After adding your images, they will automatically be displayed in the gallery.
